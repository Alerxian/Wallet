# 04 - 功能矩阵（对齐 Rabby）

## 已实现（当前版本）

### 钱包管理
- 创建钱包（12/24 词，BIP39 助记词派生地址）
- 导入钱包（助记词/私钥导入并派生地址）
- 多钱包列表
- 钱包切换/删除

### 资产与网络
- 资产列表展示
- 代币隐藏/显示
- 预置多网络（Ethereum/Arbitrum/Polygon）
- 自定义网络添加
- 网络启停与切换

### 交易
- 发送页（地址格式校验、数量校验、备注）
- 主币真实交易广播（RPC）
- 接收页（地址二维码 + 复制）
- 交易历史（含 Gas、Nonce、状态）

### 真实链路（Phase 1）
- 主币余额实时同步（RPC）
- 价格实时同步（CoinGecko）
- 下拉刷新触发链上同步

### dApp
- dApp 连接会话（模拟）
- 书签管理
- 历史记录
- 内置浏览器（WebView，URL 导航、前进后退、刷新、收藏）

### DeFi/NFT/Portfolio/硬件钱包
- DeFi 仓位展示
- NFT 网格展示
- Portfolio 分布展示
- 硬件钱包集成框架页面（待 SDK）

## 与 Rabby 的差距（下一步）

1. ERC-20 余额查询与发送
2. 多链交易历史真实同步
3. 授权扫描、交易模拟、风险评分
4. WalletConnect v2 正式协议栈
5. dApp 权限细粒度管理（账户/链/方法）
